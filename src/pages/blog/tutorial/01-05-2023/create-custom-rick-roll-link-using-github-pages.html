  • Have Github account. If you don't have one, go ahead and set it up.
{% include heading.html level="2" text="Step by step guide" %}
  1. Ok, you're logged in on home page of github.com. The first thing you need to do is click on your profile logo in the top right corner ("1" in the picture) of the page and click on select "Your repositories" in the pop-up menu ("2" in the picture). {% include blog_post_image.html src = "./first-step.png" width = "1200px" %}
  2. Now, on this page should be a green button that has "New" written on it ("3" in the picture). Go ahead and click on it. {% include blog_post_image.html src = "./second-step.png" width = "1200px" %}
  3. Now, you're creating a repository. Make sure that your settings are identical to the ones shown below. Especiallly the part where you make your repository "Public", otherwise it won't work (without Pro/Enterprise account). In the textbox repository name, write the name that you want your link to have. For example, I have "link-name", that means that my final link would look like this: https://tdiblik.github.io/link-name. This link is composed of: https://YOUR_USERNAME.github.io/REPOSITORY_NAME. After you're done with it you can go ahead and click on the "Create repository" button. {% include blog_post_image.html src = "./third-step.png" width = "1200px" %}
  4. After that you'll be redirected to another page, this page home page of your repository. In here, you need to click on "creating a new file" {% include blog_post_image.html src = "./forth-step.png" width = "1200px" %}
  5. That should redirect you to another page, where you need to make sure that you input the following information into the circled sections. To the first circled section, write: _config.yml. And to the second circled section, write: theme: jekyll-theme-cayman. After you've done that, continue by clicking the "Commit changes" button and then confirm it inside the pop-up. {% include blog_post_image.html src = "./fifth-step.png" width = "1200px" %} {% include blog_post_image.html src = "./sixth-step.png" width = "1200px" %}
  6. Once you've confirmed the commit, you will be once again redirected to the repostiory home page. From here, you want to create another new file called index.html, insert following code into it and commit it (following pictures as instructions). 
    <!DOCTYPE html>
<meta charset="utf-8" />
<title>Oh no no no :D</title>
<meta
  http-equiv="refresh"
  content="0; URL=https://www.youtube.com/watch?v=dQw4w9WgXcQ"
/>
    {% include blog_post_image.html src = "./seventh-step.png" width = "1200px" %} {% include blog_post_image.html src = "./eight-step.png" width = "1200px" %} {% include blog_post_image.html src = "./nineth-step.png" width = "1200px" %}
  7. Right now, we need to activate github pages and we're all done. You can do that by going into Settings -> Pages -> Branch, then select master or main (depending on your settings) and click on "Save". {% include blog_post_image.html src = "./final-step.png" width = "1200px" %}
  8. All you need to do is wait for a minute and then try your link. You can either construct it yourself following the pattern https://YOUR_USERNAME.github.io/REPOSITORY_NAME, or you can see it in the same "Pages" settings page. {% include blog_post_image.html src = "./where-to-see-link.png" width = "1200px" %}

And that's it! If you're interested in learning how it works or want to use different video instead of rick roll, you can read about it in the next blog post.
